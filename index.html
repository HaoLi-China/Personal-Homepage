PROJECTS

Pr2 scans the room with KinectFusion automatically

In this project, the pr2 can grasp a Kinect and move to scan the indoor scenes with KinectFusion automatically.We use socket communications to exchange the data between ros and windows

Pr2 opens door automatically

In this project, we drive pr2 to the front of a cabinet, the pr2 can detect the handle and open the door automatically, and then closes it.

Splendour(HTML5 version)

A network webgame based on HTML5, Device APIs, webSocket, webGL and so on. In the game, players can catch all kinds of butterflies by their fingers. We use camera to recognize the gesture of players to do this. The game supports both single player and double players on the Internet.

Splendour(Android version)

An android game controlled by gravity and voice. In the game, players can catch all kinds of butterflies.

SHUN

An equipment file sharing software based on DLNA and UPNP on Meego operating system. We use streaming media technology to achieve file sharing easily from a certain device files to another device in a seamless transfer.

Cool cloud album

A cloud album android application software with functions including taking photos, editing photos, view photos, sharing and distributing. We added some special functions, such as remote taking photos, voice recording for pictures, voice search albums, manual rendering and so on.

RushHour

An adventure game with 3 levels based on android.

HouseKeeper

An intelligent housekeeper system which integrates wireless communications, environmental monitoring, real-time positioning, remote control, safety alarm and so on.

OurBall

An android game controlled by gravity. In the game, players must avoid the ball roll in a hole, and the ball can roll from one device to the other device. The game supports both single player and double players.
